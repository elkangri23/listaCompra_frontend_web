# Tokens de diseño

La base de nuestro sistema de diseño está construida sobre tokens semánticos que permiten escalar la interfaz en web y
plataformas futuras. Los valores se mantienen sincronizados con Tailwind y los componentes de la librería UI.

## Colores

<Canvas>
  <div className="grid gap-4 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4">
    {Object.entries(designTokens.colors).map(([token, value]) => (
      <div key={token} className="rounded-lg border bg-card p-4 shadow-sm">
        <div className="flex items-center justify-between text-sm font-medium">
          <span>{token}</span>
          <code className="text-xs text-muted-foreground">{value}</code>
        </div>
        <div className="mt-4 h-16 rounded-md border" style={{ background: value }} />
      </div>
    ))}
  </div>
</Canvas>
